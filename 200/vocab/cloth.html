Common Clothes Words

This page gives you some common words for types of clothing.

   This is a shirt.
  This is a blouse.
  This is a suit.
  This is a dress.
  This is a skirt.
  These are shorts.
  These are pants.
  This is a hat.
  These are socks.
  This is a sweater.

When you are sure that you understand these words, you can continue with the exercises.
