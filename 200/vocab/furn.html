Common Furniture Words

This page gives you some common words that describe furniture.

   This is a sofa.
  This is a chair.
  This is a mirror.
  This is a bookcase.
  This is a dresser.
  This is a table.
  This is a bed.
  This is a television.
  This is a video player.

When you are sure that you understand these words, you can continue with the exercises.
